# Hisaab Kitaab 💸

A simple and powerful web application to track your daily financial transactions. Manage "Lena-Dena" (debts and credits) with friends and family effortlessly.

## ✨ Features

- **📱 PWA Support**: Installable as a native app on your mobile device via "Add to Home Screen".
- **💾 Local Data Storage**: All your data is stored locally in your browser (LocalStorage). No server or database required.
- **🔙 Native Back Navigation**: Mobile back button closes active popups and modals first, ensuring a smooth app-like experience.
- **🖐️ Swipe-to-Close**: Intuitive swipe-down gestures to close bottom drawers and modals.
- **📄 PDF Export**: Generate and download a professional PDF summary of any member's transaction history.
- **📚 Multiple Books**: Create separate books for different events or groups (e.g., Office, Home, Goa Trip).
- **🌓 Dark/Light Mode**: Full support for system themes with a beautiful dark mode.
- **👤 Personalized Greeting**: Custom welcome message and user name on the dashboard.
- **🔍 Quick Search**: Instantly find members by name or email.
- **📊 Summary Dashboard**: Clear overview of total amounts to receive (Lena) and total amounts to pay (Dena).
- **👻 Modern UI**: A vibrant, clean interface with glassmorphism effects and colorful gradients.

## 🚀 Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **PDF Generation**: jsPDF
- **State Management**: React Hooks (useState, useEffect, useCallback)
- **Deployment**: Optimized for Vercel/Static Hosting

## 🛠️ Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```
2. **Run Development Server**:
   ```bash
   npm run dev
   ```
3. **Build for Production**:
   ```bash
   npm run build
   ```

## 🔒 Privacy & Security

Your data is 100% private. Since the application does not use a central server, your financial records never leave your device.
